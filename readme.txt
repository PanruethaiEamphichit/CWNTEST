How to prepare .onchain Source File (For enrollMobileId)
1. Create .onchain file from .exe
2. Put .onchain file to folder XXX
3. Rename .onchain file to TC No.
4. Set path of folder XXX in variable.robot file - variable name: ${onchainJson_Path}

path: /CWNTEST/mobileId/Tests>
robot [testcase name]
